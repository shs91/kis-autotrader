# 시스템 아키텍처 다이어그램 (Excalidraw / tldraw)

## 옵션 E: Excalidraw용 구조 가이드

Excalidraw(https://excalidraw.com)나 tldraw(https://www.tldraw.com)는
손그림 스타일의 다이어그램을 만들 수 있어 발표용으로 인기가 높습니다.

아래 구조를 참고해서 Excalidraw에서 직접 그리면 됩니다.
