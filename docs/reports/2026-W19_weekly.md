## 중기 아키텍처 논의

### 1. MACD 영구 비활성화 — 일봉 데이터 윈도우 한계

KIS API 일봉 조회가 30건만 반환하는 한계로, MACD 전략(필요 36건 = slow 26 + signal 9 + 1)이 **시스템 가동 이래 단 한 번도 정상 평가되지 않음**. SIGNAL_SKIP 메트릭의 vote_meta 분석 결과, 모든 사이클에서 MACD 투표가 `guard_reason=insufficient_length`로 차단되어 앙상블이 사실상 3개 전략(MA/RSI/Bollinger)에만 의존.

→ **앙상블 다양성 부족** (시그널 유형 분산 = 0). 단일 전략 의존도가 상승하여 시장 환경 변화 시 일관된 오판 가능성.

### 2. HOLD 우세 구조 — 30일 윈도우의 표현력 한계

SIGNAL_SKIP 표본(05-08, 6,638건) 분석:
- RSI(14): 평가된 종목들의 last_rsi 38~64 → 모두 중립(과매수/과매도 임계 미도달)
- Bollinger(20,2.0): last_percent_b 0.07~0.76 → 중립
- 4개 전략 중 3개가 거의 항상 HOLD를 반환하는 구조

30일 데이터로는 추세 반전·돌파 시점이 드물고, 평균 기간을 짧게 잡으면 노이즈가 증가. **데이터 윈도우 확장(60~120일)이 지표 표현력 회복의 전제 조건**.

### 3. ETF/Q-code 필터 누수 — 코드 명백한 결함

스크리닝 결과(05-06~08, 5,094건)에서 KODEX 인버스/레버리지 ETF가 다수 잔존:
- 252670 (KODEX 200선물인버스2X): 721건
- 114800 (KODEX 인버스): 715건
- 252710 (KODEX 코스닥150선물인버스): 715건
- 069500 (KODEX 200): 676건
- 396500/379800 (TIGER/KODEX 미국 ETF): 1,297건
- Q530036/Q550043 등 Q-prefix 코드: 813건

`src/strategy/screener.py::_is_etf_etn`이 `stock_name`의 키워드(KODEX/TIGER/...)를 검사하나, KIS volume_rank 응답에서 stock_name이 stock_code와 동일하게 반환되는 경우 키워드 매칭 실패. `Q*` 코드 필터도 결과상 무효 (워커 외부 경로로 유입 추정). **05-07 acted 시그널 1499건 중 1498건이 252670/252710** — 사실상 전 주 매매 시그널이 잘못된 종목에서 생성됨.

### 4. act_rate 메트릭의 구조적 왜곡

W19 시그널 7,631건의 분포:
- SELL(미보유 종목, `skip_reason=sell_without_position`): 67% — 구조적으로 실행 불가
- BUY(이미 보유): 33% — `action_taken=False`
- 실제 BUY 실행: 2건 (0.03%)

→ 현재 act_rate(19.7%)는 ETF 시그널 잡음(05-07 1,498건)으로 부풀려진 수치. **시그널 메트릭 재정의 필요** (예: 신규 BUY 시그널/평가 기회 비율).
